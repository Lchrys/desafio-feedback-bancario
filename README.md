# Desafio Criativo: Extraindo Insights do Feedback de Clientes Bancários

Desafio desenvolvido como parte da trilha de IA da DIO, com foco na construção de prompts estruturados para análise de feedbacks de clientes bancários.

## Objetivo

Criar um prompt capaz de orientar uma IA a extrair insights claros e acionáveis a partir de comentários de clientes de um banco digital.

## Estrutura do Prompt

O prompt final foi construído em três etapas:

1. Definição da intenção — o que a IA deve produzir, para quem e com qual finalidade
2. Contexto e restrições — dados disponíveis, critérios de análise e cuidados com informações sensíveis
3. Refinamento — unificação de todos os elementos em um comando único e claro

## Arquivo

- `prompt-feedback-bancario.md` — prompt final pronto para uso

## Tecnologias e conceitos aplicados

- Engenharia de prompts
- Análise de experiência do cliente
- Boas práticas com dados sensíveis em contexto bancário
